# Alishka&Madishka Show - Personal Website

A beautiful, modern personal website built with HTML, CSS (TailwindCSS), and JavaScript. Features responsive design, smooth animations, and interactive elements.

## 🚀 Features

- **Modern UI Design**: Clean, professional design with gradient accents
- **Fully Responsive**: Works perfectly on all devices (desktop, tablet, mobile)
- **Smooth Animations**: Parallax effects, fade-ins, and micro-interactions
- **Interactive Navigation**: Sticky header with smooth scrolling
- **Contact Form**: Functional contact form with validation
- **Dark Mode**: Toggle between light and dark themes
- **Particle Effects**: Subtle floating particle animation
- **SEO Optimized**: Semantic HTML5 structure with proper meta tags
- **Accessibility**: WCAG compliant with ARIA labels and keyboard navigation

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for better SEO and accessibility
- **TailwindCSS**: Utility-first CSS framework for rapid styling
- **JavaScript (Vanilla)**: Modern ES6+ for interactions and animations
- **Font Awesome**: Icon library for social links and UI elements
- **Google Fonts**: Inter font for typography

## 🎨 Customization

### Personal Information
Update the following in `index.html`:
- Your name in the hero section and navigation
- About section content
- Project details and links
- Skills and expertise
- Contact information and social links

### Color Scheme
The website uses a purple gradient theme. Modify the CSS variables in the `<style>` section:
```css
.gradient-text {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

.hero-gradient {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

### Typography
The website uses the Inter font family. You can change this by modifying the Google Fonts import.

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in your preferred web browser
3. **Customize** the content to match your personal information

No build process or dependencies required - it's a static website!

The contact form is set up for client-side validation. To make it fully functional, you'll need to:
1. Set up a backend service (Node.js, PHP, etc.)
2. Integrate with a form service (Formspree, Netlify Forms, etc.)
3. Update the form submission handler in `script.js`
